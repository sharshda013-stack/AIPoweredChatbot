# 🤖 AI Powered Chatbot

## 📌 Overview
AI Powered Chatbot is a simple web-based chatbot application built using Python and Flask. It provides instant responses to user queries using predefined logic.

---

## ✨ Features
- 💬 Interactive chat interface  
- ⚡ Instant responses  
- 🌐 Web-based chatbot using Flask  
- 🎯 Simple and beginner-friendly project  

---

## 🛠️ Technologies Used
- Python  
- Flask  
- HTML  
- CSS

## ⚙️ How to Run This Project

1. Install dependencies:

pip install -r requirements.txt


2. Run the app:

python app.py


3. Open in browser:

http://127.0.0.1:5000/
